# FreeCAD Developer Hub

Welcome to the FreeCAD Developer Hub - your central resource for contributing to FreeCAD development.

- [Developer Handbook](https://freecad.github.io/DevelopersHandbook)
- [Source Code](https://github.com/FreeCAD/FreeCAD)
- [Issue Tracker](https://github.com/FreeCAD/FreeCAD/issues)

## Getting Started

1. **Set up your environment**: Follow the [build instructions](https://freecad.github.io/DevelopersHandbook/getting-started/) for your platform
2. **Pick an issue**: Browse [good first issues](https://github.com/FreeCAD/FreeCAD/contribute)
3. **Join the community**: Introduce yourself on the [forum](https://forum.freecad.org)

## Documentation Structure

- `/DeveloperHub/` - Central developer resources (this page)
- `/Contributing/` - How to contribute
- `/Modules/` - Module-specific developer docs
- `/API/` - API references

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a Pull Request

See the [Contributing Guide](https://github.com/FreeCAD/FreeCAD/blob/main/CONTRIBUTING.md) for details.
